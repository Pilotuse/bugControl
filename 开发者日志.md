## 修改角色定位
改造时间：2021年2月12日17:19:49
修改角色权限定位，将数字定位角色改成如下：

| 角色标志词 | 对应显示路由 | 菜单权限                                   | 是否支持个性化菜单 |
| ---------- | ------------ | ------------------------------------------ | ------------------ |
| 超级管理员 | admin        | 全部权限                                   | 否                 |
| PM客户经理 | pm           | 工作台，项目，任务，文档，统计，成员，设置 | 否                 |
| 开发       | developer    | 工作台，项目，任务，文档，统计，设置       | 是                 |
| 测试       | tester       | 工作台，项目，任务，文档，版本，统计，设置 | 否                 |
| 未定义用户 | undefined    | 无                                         | 否                 |


## 修改菜单项目
不再区分用户群体，根据角色去调用接口后返回数据进行渲染
